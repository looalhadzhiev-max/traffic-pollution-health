Problem Statement



Urban traffic is one of the major contributors to local air pollution. Pollutants such as PM2.5, PM10, and NO₂ are associated with negative health outcomes, especially respiratory and cardiovascular risks.



The main research question is: To what extent does urban traffic contribute to air pollution, and how can this pollution be translated into estimated health risk?



Objectives



The project aims to:



1. Collect and combine data from multiple independent sources.
2. Clean, validate, and prepare traffic, pollution, weather, and health-related data.
3. Analyze the relationship between traffic intensity and air pollution.
4. Estimate health risk based on pollution exposure.
5. Build and evaluate a multi-step statistical model.



Planned Methods



The project will use:



* Exploratory data analysis
* Correlation analysis
* Linear regression
* Multiple regression with weather controls
* Lagged feature analysis
* Estimated health-risk modeling
* Model diagnostics and limitations discussion



Data Sources



The project uses two independent data sources:



1\. UCI Air Quality Dataset  

&#x20;  Hourly urban air-quality and meteorological measurements.



2\. World Bank Indicator SH.STA.AIRP.P5  

&#x20;  Mortality rate attributed to household and ambient air pollution, age-standardized, per 100,000 population.



A WHO AirQ+ resource is also used as a methodological reference for health-risk interpretation.



All external datasets and scientific references will be documented in the notebook and final report.



Academic Integrity



All datasets, articles, formulas, and code references used in this project will be properly cited. The analysis and implementation will be original unless explicitly stated otherwise.
